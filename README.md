# Proyecto Canvas Sol

## Descripción
Aplicación web desarrollada con HTML, CSS, JavaScript y Canvas API 2D que replica un automóvil deportivo mediante programación.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Canvas API

## Características
- Más de 30 figuras básicas
- Imagen original y dibujo programado
- Navbar y Footer
- Diseño moderno
- Código documentado
